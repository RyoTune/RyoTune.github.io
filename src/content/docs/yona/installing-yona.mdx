1. Download and install **.NET 8 Runtime** for your system: [https://dotnet.microsoft.com/en-us/download/dotnet/8.0](https://dotnet.microsoft.com/en-us/download/dotnet/8.0)

<Image src={img_dl_net8} alt=".NET 8 Runtime Download" />

2. Download the latest version of **Yona** on **Github** for your system: [https://github.com/RyoTune/Yona/releases](https://github.com/RyoTune/Yona/releases)

<Image src={img_dl_yona} alt="Yona Download" />

3. Extract **Yona** from the downloaded file.

<Image src={img_extract_yona} alt="Extract Yona with 7-Zip" />

4. Start **Yona** by running `Yona.exe` on Windows or `Yona` on Linux.

<Image src={img_yona_home} alt="Yona Home Page" />
