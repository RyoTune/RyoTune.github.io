---
title: The Reloaded Console
description: Reloaded's console is an invaluable tool for making mods or fixing errors.
---

import { Image } from "astro:assets";

import img_re_home from "./images/reloaded-home.png";
import img_re_home_console from "./images/reloaded-home-console.png";
import img_re_console from "./images/reloaded-console.png";

1. Go to Reloaded's **Mod Loader Settings** page.

<Image src={img_re_home} alt="Reloaded's Mod Loader Settings page." />

2. Enable **Show Console**.

<Image
  src={img_re_home_console}
  alt="Enable Show Console on the Mod Loader Settings page."
/>

3. Starting a game will now open the **Reloaded Console** in a second window.

<Image src={img_re_console} alt="The Reloaded Console window." />
